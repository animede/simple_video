# かんたん動画 Standalone ユーザーズガイド

このドキュメントは `simple_video_app` の **Standalone版で実際に使える機能** に限定したガイドです。

## 1. 対象と前提

- 対象: `simple_video_app`（Standalone版）
- 前提: ComfyUI が `127.0.0.1:8188` で稼働
- ブラウザ: `http://127.0.0.1:8090/`

## 2. 画面の見方

- 左: 画像参照・キャラクタ関連・初期画像生成
- 右: 生成シーケンス・動画設定・シナリオ入力・実行
- 上部 `❓ Help`: クイックヘルプ / ユーザーズガイド / テクニカルガイド

## 3. 基本フロー（T2Vを先に使う）

1. `⚙️ 生成シーケンス` で T2V 系を選択
2. `📜 シナリオ入力` に内容を記述
3. 必要なら `🤖 プロンプト生成` で補助
4. `動画を生成` を実行
5. `🎬 出力動画` で結果を確認

## 4. 画像参照で動画を作る（キャラクタ系）

1. `📥 画像ドロップ` に参照画像を設定
2. `👤 キャラクタ一覧` から選択、または `📝 キャラクタ登録`
3. `📝 何を描きたい？` に初期画像プロンプトを入力
4. `初期画像を生成` 後に動画生成を実行

補足:

- `char_edit_i2i_flf` は初期画像・キャラ画像の影響が大きいです
- 生成画像はキー画像として自動反映されます

## 5. 音楽系フロー

### 5.1 T2A（テキストから音楽を生成）

1. 音楽条件（雰囲気・歌詞イメージ）を入力
2. 必要なら歌詞/タグ提案を利用
3. 音楽生成を実行

### 5.2 M2V（音楽から動画を生成）

1. 生成音源またはアップロード音源を選択
2. 動画シナリオを入力
3. `動画を生成` を実行

動画シナリオ未入力時:

- `音楽→動画` 実行時に確認ダイアログを表示
- `このまま生成する`: 歌詞/音源ベースで継続
- `シナリオを入力する`: シナリオ欄へ戻って編集

### 5.3 V2M（動画から音楽を生成）

1. 生成済み動画またはアップロード動画を選択
2. 必要なら音楽スタイル条件を設定
3. 実行して音楽合成結果を確認

## 6. エラー時の確認

- `ComfyUI /prompt failed` が出たら、エラー詳細の停止箇所を確認（`node_errors` 欄）
- 画像参照エラー時は `📥 画像ドロップ` とキャラ選択状態を再確認
- 古い `job_id` の 404 は再生成で解消する場合があります

## 7. Standalone版で未対応の機能

- Utility機能（本家向けの汎用API呼び出し）
- distributed モード
- 複数ユーザー同時運用（single-user 前提）
