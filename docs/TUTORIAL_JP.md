# かんたん動画 チュートリアル

このチュートリアルは、最短で1本の動画を作る手順です。

## 1. 起動

```bash
cd simple_video_app
./start.sh
```

ブラウザ: `http://127.0.0.1:8090/`

## 2. キャラクタ画像の生成（左カラム）

1. `📥 画像ドロップ` に参照画像を必要に応じて入れる（ref1〜ref3）
2. `👤 キャラクタ一覧` で既存キャラを選ぶ、または `📝 キャラクタ登録` で新規登録
3. `📝 何を描きたい？` に初期画像プロンプトを入力
4. `初期画像を生成` を押す

補足:

- `char_edit_i2i_flf` では、初期画像/キャラ画像が重要です
- 生成後、キー画像として自動反映されます

## 3. 生成シーケンスと動画設定（右カラム）

1. `⚙️ 生成シーケンス` を選択
2. `⚙️ 動画設定` でシーン数・長さ・必要なら詳細設定を調整

## 4. シナリオ入力

- `📜 シナリオ入力` に作りたい内容を記述
- 必要に応じて `🤖 プロンプト生成` を利用

## 5. 動画生成

1. `動画を生成` を押す
2. 中間画像確認が有効な場合、`CONTINUE` で後段へ進む
3. `🎬 出力動画` に結果が表示される

## 6. つまずいたとき

- `ComfyUI /prompt failed` が出たら、エラー詳細の「どの設定や入力で止まったか」を確認（`node_errors` 欄に原因が出ます）
- 画像参照エラー時は、`📥 画像ドロップ` とキャラ選択状態を再確認
- 古い job_id の404は、再生成すると解消されることがあります
