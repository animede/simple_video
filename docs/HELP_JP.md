# かんたん動画 クイックヘルプ

このヘルプは、**すでにアプリを開いている状態**での操作手順だけをまとめています。

## 1. T2V（テキストから動画を生成）

1. `⚙️ 生成シーケンス` で T2V 系（例: テキストから直接動画を生成）を選択
2. `📜 シナリオ入力` に作りたい内容を書く
3. 必要なら `🤖 プロンプト生成` で補助
4. `動画を生成` を押す

## 2. キャラクタ動画（画像を参照して生成）

1. `📥 画像ドロップ` に参照画像を入れる
2. `👤 キャラクタ一覧` で選択、または `📝 キャラクタ登録`
3. `📝 何を描きたい？` に初期画像プロンプトを入力
4. `初期画像を生成` を押してから動画生成へ進む

補足:

- `char_edit_i2i_flf` では初期画像・キャラ画像が特に重要です
- 生成した画像はキー画像として自動反映されます

## 3. T2A（テキストから音楽を生成）

1. 音楽生成の入力欄に、曲の雰囲気や歌詞イメージを入力
2. 必要に応じて歌詞・タグ提案を使う
3. 音楽生成を実行して、生成音源を確認

`--ace-step-url` 指定時は `🧠 Thinking`（高品質）と `✨ AI Tags`（タグ強化）が使えます。

## 4. M2V（音楽から動画を生成）

1. 生成音源またはアップロード音源を選択
2. 動画シナリオを入力（未入力時は確認ダイアログが表示）
3. `動画を生成` を実行

## 5. V2M（動画から音楽を生成）

1. 生成済み動画またはアップロード動画を選択
2. 必要に応じて音楽スタイル条件を指定
3. 実行して、動画に合成された音楽結果を確認

## 5.5 🎵 動画に音楽を追加

- 結合動画表示時に `🎵 音楽を追加` ボタンが現れます（音源がある場合のみ）
- 押すと動画に音楽トラックを合成して PV を作成

## 6. 困ったとき

- `ComfyUI /prompt failed` が出たら、エラー詳細の「どの設定や入力で止まったか」を確認（`node_errors` 欄に原因が出ます）
- 画像参照エラー時は、`📥 画像ドロップ` とキャラ選択状態を再確認
- 古い job_id の404は、再生成で解消することがあります
